Calendar

Description of Task:

Application for managing calendar with the ability to create events and search for them.

Implemented functionalities of the following:

 1)   daily, weekly and monthly overview of calendar
 2)   creating and managing events (time, place, date, participants,...)
 3)   ability to settting event recurrence (once a week, 14 days, once in month)
 4)   search for events by name and / or place
 5)   searching free time for events and moving events
 6)   exporting and importing events

Using Polymorphism:

    kinds of events: important, optional, moveable
    event time change strategy: unchangable, first available with at most X moves
    types of calendars overview (daily, weekly, monthly)
    formats of exported events
